# CNN From Scratch Using NumPy

I built a complete CNN forward pass from scratch using only NumPy.
No PyTorch, no TensorFlow, pure math only.

This is Day 1 of my journey to deeply understand neural networks 
from the ground up before using library abstractions.

Maintained by Balamuralikrishna Durukunala
MS Applied Artificial Intelligence @ Robert Morris University

---

## What I Built

Convolution operation from scratch using manual sliding window math
ReLU activation from scratch
MaxPooling from scratch
Complete CNN forward pass pipeline combining all three

---

## Results

Input image 5x5 goes through the complete pipeline:

Input (5x5) → Convolution (3x3) → ReLU (3x3) → MaxPool (2x2)

Built using only NumPy with zero deep learning libraries!
